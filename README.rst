pyBio
=====

pyBio is a toolkit for bioinformatics.

Look how easy it is to use::

    import pybio
    # Get your stuff done
    pybio.do_stuff()

Features
--------

- peptide sequence annotations
- integrated with  major biological databases
- biological mass spectrometry friendly

Installation
------------

Install pyBio by running:

    pip install project
